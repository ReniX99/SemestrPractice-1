Для запуска проекта введите следующие команды:
1. Клонировать указанный репозиторий
```{git}
git clone https://github.com/ReniX99/SemestrPractice-1
```

2. Запустить сервер (3000 порт):
```{bash}
cd back
npm install
node server.js
```

3. Запустить фронтенд (5173 порт):
```{bash}
cd front
npm install
npm run dev
```
